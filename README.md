# Simple Express Chatbot 🤖💬

A lightweight, fast, and fully functional chatbot backend built with Node.js and Express. This project demonstrates how to set up a web server to handle conversational routing and process user messages.

## 🚀 Features
* **Custom Chat Logic:** Processes incoming text and returns calculated responses.
* **REST API Architecture:** Uses standard HTTP POST requests for communication.
* **Frontend Included:** Comes with a clean, sky-blue HTML/CSS web interface.
* **Cloud Ready:** Configured with dynamic ports for seamless deployment on platforms like Render.
